# Training Program

Aplicación de programa de entrenamiento construida con React + Vite.
Incluye seguimiento de fases, sesiones, gráficas de progreso y temporizador.

## Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [lucide-react](https://lucide.dev/) — iconos
- [recharts](https://recharts.org/) — gráficas

## Desarrollo

```bash
npm install
npm run dev
```

La app queda disponible en `http://localhost:5173`.

## Build de producción

```bash
npm run build
npm run preview
```

El código de la aplicación vive en [src/App.jsx](src/App.jsx).
